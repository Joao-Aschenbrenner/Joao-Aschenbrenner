# Logos das Empresas

Esta pasta contém os logos das empresas mencionadas na seção de experiência profissional.

## Arquivos necessários:

1. **santa-casa-logo.png** - Logo da Santa Casa de Misericórdia de Taquarituba
2. **rubi-logo.png** - Logo da Rubi Comércio Eletrônico / Rubi Presentes
3. **equipe-logo.png** - Logo da Equipe Informática

## Instruções:

Adicione as imagens dos logos nesta pasta com os nomes especificados acima.
As imagens devem estar em formato PNG e ter fundo transparente ou adaptado para o tema escuro do perfil.
